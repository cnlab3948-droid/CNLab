Research Areas

주요 연구 분야

CNLab은 인간의 인지 메커니즘을 다양한 관점에서 연구합니다.

❤️

정서 (Emotion)

정서가 인지 과정에 미치는 영향을 연구합니다. 감정 조절, 정서적 주의 편향, 정서와 기억의 상호작용 등을 탐구합니다.

💬

언어 (Language)

언어 처리의 인지적 메커니즘을 연구합니다. 언어 이해, 산출, 읽기 과정에서의 인지 기능을 탐구합니다.

🎯

의사결정 (Decision Making)

인간의 판단과 의사결정 과정을 연구합니다. 불확실성 하의 선택, 위험 지각, 인지 편향 등을 탐구합니다.

🧠

작업기억 (Working Memory)

정보의 일시적 저장과 처리를 담당하는 작업기억의 용량, 정밀도, 갱신 과정을 탐구합니다. 시각 작업기억을 중심으로 연구합니다.

Experiment

온라인 실험

웹 브라우저에서 반응시간(RT)을 측정하는 시범 실험입니다. performance.now() API를 사용하여 밀리초 단위 정밀도로 측정합니다.

🟢 단순 반응시간 과제

Simple Reaction Time Task

📋 실험 방법:
1. 시작 버튼을 누르면 화면 중앙에 + 기호가 나타납니다.
2. 잠시 후 초록색 원이 나타나면 최대한 빨리 스페이스바를 누르세요.
3. 총 10회 시행을 완료한 후 결과가 표시됩니다.

🔴🔵 선택 반응시간 과제

Choice Reaction Time Task

📋 실험 방법:
1. 시작 버튼을 누르면 화면 중앙에 + 기호가 나타납니다.
2. 빨간색 원이 나타나면 → F 키를 누르세요.
3. 파란색 원이 나타나면 → J 키를 누르세요.
4. 총 20회 시행을 완료한 후 결과가 표시됩니다.

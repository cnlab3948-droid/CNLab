📝 CNLab 홈페이지 직접 업데이트 통합 가이드

이 문서에서는 개발자나 저의 도움 없이도 연구실 인원 누구나 직접 공지사항, 구성원, 논문, 연구, 설문조사, 일정 내용을 아주 쉽게 수정하고 업데이트하는 방법을 안내합니다.
모든 작업은 프로그램 설치 없이 GitHub 웹사이트에서 바로 진행됩니다.

1. 📢 공지사항 추가/수정하기

공지사항 전용 파일인 announcements.json에 규칙에 맞게 짧은 텍스트만 덧붙이면 화면에 반영됩니다.

  1. [CNLab 저장소]에서 data 폴더 클릭 ➔ announcements.json 파일 클릭
  2. 열린 파일 우측 상단의 펜 모양 ✏️ 아이콘(Edit) 클릭
  3. 기존 글 양식을 복사하여 새로운 칸에 붙여넣고 수정합니다.
{
  "id": 6,
  "category": "알림",
  "title": "여기에 제목을 적으세요",
  "date": "2026-05-01",
  "summary": "간단한 요약을 적습니다.",
  "content": "이곳에 상세 내용을 적습니다.\n\n줄바꿈은 \\n 으로 적어야 합니다.",
  "pinned": true
}
주의: 중괄호 { }와 쉼표(,) 위치를 꼭 맞춰주세요. 가장 마지막 게시글의 괄호 } 뒤에는 쉼표가 없어야 문법 에러가 나지 않습니다.

2. 일반 텍스트 정보 바꾸기 (👨‍🔬구성원 / 📄논문 / 🔬연구 / 📊설문)

이 정보들은 메인 뼈대인 index.html 파일 안에 모두 담겨 있습니다. 눈으로는 찾기 어려우니 단어 찾기(Ctrl + F) 기능을 씁니다!

  1. GitHub 첫 화면에서 index.html 파일 클릭 ➔ 펜 모양 아이콘 ✏️ 클릭
  2. 윈도우 기준 Ctrl + F를 눌러 바꾸고 싶은 핵심 단어를 검색합니다.

🔬 [연구 탭] 연구 키워드 / 진행 중 프로젝트 수정

  • Ctrl + F 검색어: "연구 키워드" 또는 "진행 중 프로젝트"
  • 해당 줄을 찾으셨다면, 까만색 텍스트로 적힌 기존 내용을 지우고 새로운 내용으로 수정합니다.

📊 [설문 탭] 설문조사 링크 및 내용 바꾸기 (Google Forms)

  • Ctrl + F 검색어: "설문조사" 또는 "survey-card"
  • 설문조사 이름, 마감 기한 등 한글 텍스트를 새로운 내용으로 교체합니다.
  • 💡 구글 폼 링크 교체: 코드를 보다 보면 아래와 같은 긴 부분이 나옵니다.
    <button onclick="window.open('여기_URL_붙여넣기', '_blank')">
    저 따옴표 안에 있는 URL 주소를 연구실의 새 구글 폼 링크로 갈아치워 줍니다.
⚠️ 필수 주의사항: 절대 자잘한 꺾쇠 코드들(<p>, <div class="..."> 등)은 한 글자도 건드리지 마세요! 화면 UI가 박살납니다. 꺾쇠 바깥에 있는 까만 글자나 주소만 쏙 골라서 수정하세요.

3. 🗓️ [일정 탭] 구글 캘린더 연동시키기

연구실 공용 아이디로 만든 구글 캘린더가 갱신되어 홈페이지에 노출시키고 싶다면 아래 과정을 따릅니다.

  1. index.html 페이지 편집 모드에서 Ctrl + F 로 "schedule__calendar-wrapper"를 검색합니다.
  2. 바로 아래 줄 쯤에 숨김 처리된(초록색) 형태의 코드가 보입니다.
    <!-- <iframe src="https://calendar.google.com/..."> </iframe> -->
  3. 맨 앞과 맨 뒤에 감싸고 있는 주석 기호 화살표들( <!-- 와 --> )을 백스페이스로 지워줍니다.
  4. src="..." 안에 있는 기존 가짜 주소를 지우고 연구실의 구글 캘린더 ID 또는 공유용 Embed 주소를 붙여넣습니다.

4. 💾 가장 중요한 '저장 및 반영' (모든 작업 공통)

GitHub에서는 파일만 고친다고 홈페이지가 알아서 바뀌지 않습니다. 완료 도장을 쾅 찍어주어야('Commit') 비로소 인터넷에 배포됩니다.

  1. 파일 수정 화면 우측 상단의 초록색 [Commit changes...] 버튼 클릭
  2. 조그만 팝업 창 메모란에 "초당 논문 1편 추가", "설문조사 링크 수정" 등 오늘 작업한 내역 메모
  3. 그 아래의 초록색 [Commit changes] 확인 버튼 꾹 누르기

🎉 수고하셨습니다!

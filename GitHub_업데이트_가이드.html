🚀 CNLab 홈페이지 운영 가이드 (구글 시트 연동)

💡 핵심 원칙: 본 홈페이지는 깃허브(GitHub) 코드를 수정할 필요 없이 오직 '구글 시트' 하나로 100% 자동 통제됩니다. 시트에 글을 적고 기다리면, 그 즉시 홈페이지에 반영됩니다. (개발 지식 불필요)

📑 1. 구글 시트 기본 구조

구글 시트 하단에 반드시 네 가지 탭(시트) 이름이 딱 맞게 존재해야 합니다.

  1. 실험기록 : 온라인 실험 참여자들의 데이터들이 컴퓨터에 의해 자동으로 찍히는 곳입니다. (건들지 마세요)
  2. 공지사항 : 홈페이지의 'News' 섹션을 통제합니다.
  3. 설문 : 홈페이지의 'Survey' (설문조사) 카드를 통제합니다.
  4. 연구 : 홈페이지의 'Research' (진행 중인 프로젝트) 영역을 통제합니다.
⚠️ 중요 주의사항:
각 탭의 맨 첫 번째 줄(1행)은 컴퓨터가 데이터를 분류하는 기준인 헤더(제목표시줄)입니다. 절대 첫 줄의 영어 철자를 지우거나 한글로 번역하시면 안 됩니다! (2번째 줄부터 글을 자유롭게 쓰시면 됩니다.)

📢 2. [공지사항] 탭 작성법

공지사항의 새 글을 올리려면 2번 줄부터 차례대로 정보를 채워주세요.

id (순번) category (분류) title (제목) date (날짜) summary (요약) content (상세 내용) pinned (상단 고정)
1 세미나 2026년도 심리언어학회 2026-04-21 2026.05.15-16 심리언어학회 참여 2026년 심리언어학회
🗓️ 일정: 2026.05.15
📍 장소: 서강대학교		 true
2 알림 실험 피험자 모집 안내 2026-05-10 간단한 내용 요약글 실험 피험자를 상시 모집합니다. false
  • id: 1, 2, 3... 순서대로 부여합니다 (중복 안 됨).
  • content: 글 안에서 줄바꿈(엔터)을 하려면 엑셀 셀 안에서 Alt + Enter 키를 누르세요.
  • pinned: true라고 적으면 중요 공지로써 맨 위 파란색 바탕에 고정됩니다. 일반 공지는 false.

📝 3. [설문] 탭 작성법

모집할 설문이 있을 때 추가하며, 마감된 설문은 진행 상태를 변경해 주시면 홈페이지 버튼이 '마감됨'으로 잠깁니다.

status (상태) title (제목) description (설명) time (소요시간) date (마감일시) link (구글폼 등 링크)
진행중 사진 기반 문장 서술 과제 사진 속 장면을 상상해서 짧은 글로 쓰는 실험입니다. 약 25분 ~2026.05.31 https://pse-983.pages.dev/
종료됨 주의력 자기보고 설문 주의력 성향을 측정하는 인지 설문 약 10분 조기 종료 https://docs.google.com/...
  • status: 반드시 진행중 또는 종료됨 둘 중 하나만 적어야 색상 뱃지와 버튼 기능이 맞게 작동합니다.
  • 설문 카드를 완전히 홈페이지에서 안 보이게 하려면 그냥 그 줄을 아예 지워버리시면 됩니다.

홈페이지의 '진행 중 프로젝트(Ongoing Projects)' 영역에 노출될 연구 카드를 생성합니다.
